BearMaps

BearMaps is a locations and route-finding web-app that gives driving directions in the city of Berkeley

Features

Turn-by-turn navigation: Get driving directions from one point to another
Search and autocomplete: Search for places in Berkeley, aided by autocomplete
Resizeable: Zoom the map in or out to different dimensions

Implementation

Languages: Java, HTML/CSS, JavaScript
Frameworks: IntelliJ IDEA, Maven

For BearMaps, my role consisted chiefly of implementing the back-end web server powering the API that the front-end - which was provided (as part of CS 61B, Berkeley's data structures class) - makes requests to. This was an individual project, and I worked extensively with Java.

The map itself comprised of different images at different resolutions, and I built the rasterer that stitched images together depending on the user's selected area (query box) and resolution. I parsed the XML file containing the dataset from Open Street Map to populate different locations on the map, and stored this in a Graph, with each vertex representing an intersection and each edge, a road.

For the autocomplete feature, I used a modified Trie which I built from the ground-up (with assistance from Lim Mingjun) to optimize prefix-matching operations. Lastly, the route-finding feature was built using the A* Search Algorithm, which finds the shortest path between two vertices in a Graph.
