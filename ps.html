Puzzle Solver

A rudimentary artificial intelligence that cracks puzzles

Image credits: CS61B Spring 2018

Features

Intelligent Puzzle Solving: Given an initial and final (Goal) state, Puzzle Solver is able to return the most efficient (shortest) sequence of steps to solve the puzzle.

Implementation

Languages: Java

Written entirely in Java, Puzzle Solver uses Best-First Search and the A* Search Algorithm.

Essentially, this implementation uses a Priority Queue, and the idea is that for every puzzle state that's possible, there's an equivalent world-state: a wrapper class contains the puzzle state together with salient information, such as what world-states came before it and what comes after.

These world-states are enqueued onto the Priority Queue, and dequeued based on a heuristic which represents the distance between this world-state and the goal state. Distance, in this case, refers to the number of moves needed to get to the final state.
